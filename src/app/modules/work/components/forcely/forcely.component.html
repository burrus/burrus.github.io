Forcely

Forcely provides code enforcement officers with a quick and easy way to log their day to day activities.

This is a company that I started in order to reduce the amount of inefficiencies in local government. I conducted research, as well as designed and built, the entire app myself using Google’s Firestore as the back-end. Currently I have one city using the app.

I’m also currently working on an angular web dashboard for Forcely to accompany the iOS app. This will allow city supervisors to gain insights into the activities their code enforcement officers are logging.

Send me an email if you’d like to hear more about why I started this company and how it’s solving problems in local government or if you’d like a test account to try out the app.
